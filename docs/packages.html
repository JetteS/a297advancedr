6.2 Quick start

In my first package, I just put some functions I used again and again in my work.

  1. Create an RStudio project (not a package). Here I advise you to create a new project in GitHub and then to clone it as an RStudio project. It is a good practice to put all your (public) stuff on GitHub (as we learned in section 2.3)
